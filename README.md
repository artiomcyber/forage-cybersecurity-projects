## 👋 About This Repository

This repository contains my completed **cybersecurity virtual experience simulations** from [Forage](https://www.theforage.com/) — an industry-leading platform that partners with top global companies to offer real-world job simulations.

Each project folder contains:
- A full **write-up** of tasks completed
- My **methodology and analysis**
- **Key findings and takeaways**
- Completion **certificate**

These simulations mirror actual tasks performed by cybersecurity analysts at some of the world's most respected organizations — giving hands-on, practical experience in a real corporate security context.
